### 2️⃣ Setup Voice System
```

Jalanin command ini buat setup voice channel automation.

```
!menu

## 💻 Installation & Development

### Prerequisites
- Node.js v16+
- Discord Bot Token
- Git

### Clone Repository
```bash
git clone https://github.com/anos-rgb/anos_dc.git
cd anos_dc
npm install
```

### Run Bot
```bash
npm start
```
